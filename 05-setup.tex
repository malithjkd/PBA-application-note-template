\begin{cautionbox}
  Verify the signal conditioner output voltage is within the ACS AIN range
  (±\SI{10}{\volt}) before connecting.  Overvoltage will damage the
  analogue input permanently.
\end{cautionbox}

\begin{center}
\begin{tabular}{@{} l l l @{}}
  \toprule
  \textbf{Signal Conditioner pin} & \textbf{ACS AIN pin} & \textbf{Wire colour (PBA std.)} \\
  \midrule
  \texttt{OUT+} & \texttt{AIN1+} & Red \\
  \rowcolor{PBALightGray}
  \texttt{OUT-} & \texttt{AIN1-} & Black \\
  \texttt{SHLD} & GND (conditioner chassis) & Bare / Green \\
  \bottomrule
\end{tabular}
\end{center}

After wiring, use a multimeter to verify the differential voltage at the
ACS AIN terminal matches the signal conditioner output.


\subsection{ACS Controller Network Setup}
\label{sec:setup:network}

The Python logging script connects to the ACS controller via Ethernet.
Confirm the controller IP address:

\begin{steplist}
  \step Open \menu{ACS SPiiPlus MMI} on the host PC.
  \step Navigate to \menu{Settings > Communication > Ethernet}.
  \step Note the IP address (default factory: \code{10.0.0.100}).
        Update the Python script variable \code{ACS\_IP} to match
        (see \cref{app:code-python}).
  \step Confirm connectivity: \code{ping \textit{<controller-ip>}} from a
        Windows CMD prompt should reply with $<$\SI{1}{\milli\second} latency.
\end{steplist}

\begin{notebox}
  If the controller IP is unknown, press the \textbf{Reset} button on the
  ACS controller front panel for 3~seconds to restore factory defaults
  (\code{10.0.0.100}).  This does \emph{not} clear motion parameters.
\end{notebox}


\subsection{Loading the Baseline Configuration}
\label{sec:setup:config}

\begin{steplist}
  \step Copy the file \filepath{GS500\_ForceCtrl\_baseline.cfg} to the
        host PC.
  \step In the ACS MMI, navigate to \menu{File > Load Configuration}.
  \step Select \filepath{GS500\_ForceCtrl\_baseline.cfg} and click
        \textbf{Open}.
  \step The MMI status bar shows \textit{``Configuration loaded
        successfully.''} Confirm no axis fault flags are active under
        \menu{Axes > Fault Status}.
\end{steplist}

\outcome{All axis status LEDs in the ACS MMI display green.
         No fault flags are shown.}
